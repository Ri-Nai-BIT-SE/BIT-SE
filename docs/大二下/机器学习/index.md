# 机器学习

这学期很后悔选的一门课，选完之后感觉自己确实是不适合去搞机器学习相关的事情了。

自己得到的分也很低，感觉大部分原因还是出在我自己身上吧，不过这课确实不是什么好课。

## 课程评价

这门课主要在介绍几种机器学习算法，而非从零开始机器学习入门，老师讲的也一般。

很少涉及到具体实践，一般都在作理论推导。

这门课其实安排在大二下的下半学期，这个时段说实话课业压力挺小的，但是我还是没什么心情去学这门课（有点开摆了）
